# Learning Rubric (1-4)

Use this once per session or monthly summary.

## 1) Concept Understanding
- 1: Needs full guidance
- 2: Partial understanding with prompts
- 3: Explains core ideas in own words
- 4: Explains clearly and applies independently

## 2) Coding Independence
- 1: Types along with support
- 2: Completes small edits with help
- 3: Completes scoped tasks mostly independently
- 4: Plans and implements small features independently

## 3) Debugging Approach
- 1: Relies on mentor to diagnose issues
- 2: Attempts debugging with prompts
- 3: Uses print/logging and reasoning to find issues
- 4: Systematic debugging and clear explanation of fixes

## 4) Reflection and Communication
- 1: Minimal reflection
- 2: Describes what changed
- 3: Describes what changed and why
- 4: Connects code changes to simulation behavior and next steps
